# Make.com Setup Instructions for YouTube to TikTok, Instagram, Facebook Automation

## Overview
This document outlines the steps required to set up automation workflows using Make.com for content distribution from YouTube to TikTok, Instagram, and Facebook.

## Prerequisites
- A Make.com account
- Access to your YouTube channel and social media accounts (TikTok, Instagram, Facebook)
- Necessary API keys and access tokens for each platform

## Step 1: Create a New Scenario in Make.com
1. Log in to your Make.com account.
2. Click on the **Create a new scenario** button.

## Step 2: Set Up YouTube as the Trigger
1. In the new scenario, click on the **+** icon to add a module.
2. Search for **YouTube** and select it.
3. Choose the **Watch Videos** trigger.
4. Connect your YouTube account by following the authorization process.
5. Configure the trigger settings (e.g., select your channel and set the frequency). 

## Step 3: Set Up TikTok as an Action
1. Click on the **+** icon to add another module.
2. Search for **TikTok** and select it.
3. Choose the **Upload Video** action.
4. Connect your TikTok account.
5. Map the video data from the YouTube trigger to the TikTok action settings.

## Step 4: Set Up Instagram as an Action
1. Again, click on the **+** icon to add another module.
2. Search for **Instagram** and select it.
3. Choose the **Create Photo** or **Create Video** action based on your content type.
4. Connect your Instagram account.
5. Map the required data from the YouTube trigger to the Instagram action. 

## Step 5: Set Up Facebook as an Action
1. Click on the **+** icon to add the final module.
2. Search for **Facebook** and select it.
3. Choose the **Create Post** action.
4. Connect your Facebook account.
5. Map the video or post data accordingly. 

## Step 6: Testing the Workflow
1. After setting up all modules, click on the **Run Once** button to test your workflow.
2. Make sure that the video from YouTube is properly uploaded to TikTok, Instagram, and Facebook.

## Step 7: Activate the Scenario
1. Once testing is successful, click on the **Schedule** button to activate your scenario.
2. Set the desired frequency for running the automation.

## Conclusion
By following these steps, you should have a fully automated workflow for sharing your YouTube videos on TikTok, Instagram, and Facebook using Make.com.

## Troubleshooting
- If you encounter any errors, check the logs in Make.com to identify the issue.
- Ensure that all accounts are properly connected and authorized.
