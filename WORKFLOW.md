# Workflow Documentation

## Overview
This document outlines the current workflow utilized in the tajongcoding/tajong repository.

### Current Workflow Components
1. **Script and Prompt Development**:  
   - Tools: **Claude** / **Gemini**  
   - Description: Scripts and prompts for various projects are developed using Claude and Gemini.

2. **Video Generation**:  
   - Tools: **Grок** / **Canva**  
   - Description: Video content is generated using Grок and Canva, enabling efficient editing and creative designs.

3. **Voice Generation**:  
   - Tool: **11Labs**  
   - Description: Voiceovers for videos are created using 11Labs, ensuring high-quality audio output.

4. **Card News Creation**:  
   - Tools: **Canva** / **Google Slides**  
   - Description: Card news is designed with Canva and Google Slides, allowing for visually appealing presentations.

5. **YouTube Upload Process**:  
   - Description: The finalized videos are uploaded to YouTube as part of the content distribution strategy.

## Conclusion
This workflow allows for a streamlined approach to content creation and distribution, ensuring that all components are effectively leveraged to produce quality outputs.
